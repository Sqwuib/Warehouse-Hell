
# Warehouse Hell

<img width="384" height="384" alt="brad_18" src="https://github.com/user-attachments/assets/73abf61f-7342-4bd0-a9c9-ebb5cb175e0b" />

Warehouse Hell is a retro-style management and survival game built in PICO-8 using Lua.

You play as an overworked warehouse employee trying to survive each shift by collecting orders, loading your trolley, packing jobs, avoiding hazards, and dealing with an annoying boss.

<img width="384" height="384" alt="brad_19" src="https://github.com/user-attachments/assets/da92224f-2adf-4853-8336-48b422a73e42" />

The game combines fast-paced movement, resource management, humour, and escalating workplace chaos.

---

## Gameplay

Each day you must:

- Collect marked orders around the warehouse
- Carry items back to your trolley
- Move the trolley to the packing station
- Complete jobs to earn money
- Manage your energy throughout the shift
- Survive distractions, hazards, and pressure

As days progress, workloads increase and the warehouse becomes more chaotic.

---

## Features

- Retro pixel-art gameplay in PICO-8
- Order picking and packing gameplay loop
- Energy and sprint system
- Upgrade shop with daily van visits
- Forklift hazards
- Dynamic boss enemy that hunts the player
- In-game phone distraction mechanic
- Day progression and performance bonuses
- Animated UI and title screens

---

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys | Move |
| Z | Interact / Grab |
| X | Sprint |
| Z + X near trolley | Open phone |

## Project Goals

Warehouse Hell was created as a fun side project focused on designing complete gameplay systems inside the limitations of PICO-8.

The project explores:

- State management
- AI pathfinding
- Collision systems
- UI design
- Progression systems
- Performance optimisation in constrained environments
